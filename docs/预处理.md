# 预处理

广义上属于一种离线查询手段

## 常见的预处理手段

### 1. 前缀和 / 前缀异或

- **用途**：O(1) 查询区间和、区间异或
- **扩展**：二维前缀和（矩阵子区域求和）
- **变体**：差分数组（区间加减操作）

### 2. 前缀/后缀最值

- **用途**：$O(1)$ 查询某个方向上的最大值/最小值
- **典型场景**：abc463_c（后缀最大值）、买卖股票问题（前缀最小值）

### 3. 稀疏表（Sparse Table）

- **用途**：$O(1)$ 查询**任意区间**的最值（RMQ 问题）
- **代价**：$O(N \log N)$ 预处理时间和空间
- **限制**：静态数组，不支持修改

### 4. 排序 + 离散化

- **用途**：将值域压缩到 $[0, N)$，配合树状数组/线段树使用
- **典型场景**：逆序对、区间第 k 小

### 5. 因子/倍数预处理

- **用途**：预筛质数（埃氏筛/线性筛）、预处理每个数的最小质因子、因子列表
- **典型场景**：数论题、GCD 相关问题

### 6. 组合数预处理

- **用途**：预处理阶乘和逆元，$O(1)$ 算 $C(n,k) \mod p$
- **典型场景**：计数题、概率题

### 7. 图/树的预处理

- **用途**：
  - **DFS 序 / 欧拉序**：将树上问题转化为区间问题
  - **LCA 倍增**：$O(\log N)$ 查询最近公共祖先
  - **树上前缀和**：根到节点路径的权值累加

### 8. 字符串预处理

- **用途**：
  - **哈希前缀**：$O(1)$ 比较子串
  - **KMP 的 fail 数组 / Z 函数**：匹配与周期性
  - **后缀数组 + height 数组**：子串相关查询

### 9. ST 表 / 单调栈 / 单调队列

- **单调栈**：预处理每个元素左/右第一个更大/更小元素（NGE 问题）
- **单调队列**：滑动窗口最值

### 10. 倍增预处理

- **用途**：跳跃结构中 $O(\log N)$ 查询第 $k$ 步的目标
- **典型场景**：函数式跳跃、LCA、博弈

---

### 核心思想

所有预处理的本质都是一样的：**花一次 $O(N)$ 或 $O(N \log N)$ 的时间提前算好，换取每次查询从 $O(N)$ 降到 $O(1)$ 或 $O(\log N)$**。选择哪种预处理，取决于查询的类型（区间和？区间最值？祖先？匹配？）。
